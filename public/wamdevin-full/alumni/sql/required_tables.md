# Alumni Required Tables

The alumni platform expects these tables to exist in the application database:

- alumni
- alumni_profiles
- alumni_events
- alumni_event_registrations
- alumni_jobs
- alumni_job_applications
- alumni_news
- alumni_donations
- donation_campaigns
- alumni_notifications
- alumni_messages
- alumni_connections
- jwt_blacklist

Use this list with alumni/scripts/preflight.php before deployment.

If your environment uses different table names, update API queries consistently before go-live.
