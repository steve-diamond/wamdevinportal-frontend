## Display Specifications for Images

### File Requirements
- **Format**: JPG or PNG
- **Size**: 600px W × 800px H (portrait)
- **Quality**: 72dpi for web, 300dpi if needed for printing
- **File Size**: Optimize to <300KB each for fast loading
- **Color Profile**: sRGB for web consistency

### Image Styling in HTML
```css
.leadership-card .leader-image {
  position: relative;
  overflow: hidden;
  height: 300px;
}

.leadership-card .leader-image img {
  width: 100%;
  height: 100%;
  object-fit: cover;  /* Crops to fit 300px height */
  transition: transform 0.3s ease;
}
```

### Recommended Photo Setup
- **Lighting**: Professional studio or consistent natural lighting
- **Background**: Clean, neutral background (solid color or blurred)
- **Framing**: Head and shoulders or professional headshot style
- **Consistency**: Similar style and quality across all 7 images
- **Expression**: Professional, approachable, confident

## Content Management

### How to Update Leader Information
1. Open `leadership.php` in editor
2. Locate the appropriate leadership card (exco-1 through exco-7)
3. Update within the `<div class="leader-info">` section:
   - **Header**: `<h5 class="leader-title">Name</h5>`
   - **Position**: `<p class="leader-position">Title</p>`
   - **Bio**: `<p class="leader-description">Biography</p>`
4. Save file
5. Fresh page load to verify

### How to Change Images
1. Replace image files in `assets/images/exco/` folder
2. Maintain naming convention: exco-1.jpg through exco-7.jpg
3. Keep dimensions: 600×800px
4. No code changes needed - paths are already correct
